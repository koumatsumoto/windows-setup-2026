# Windows 11 クリーンインストール手順

## 1. インストール USB の作成

### 必要なもの

- Windows PC
- 16GB 以上の USB メモリ（データは消去される）

### 手順

1. Microsoft 公式ページを開く
   - 「Windows 11 をダウンロードする」ページにアクセス
   - 「インストール メディアの作成」セクションを探す

2. Media Creation Tool をダウンロード
   - 「ツールを今すぐダウンロード」をクリック

3. ツールを起動
   - ダウンロードした MediaCreationTool.exe を実行
   - ライセンス条項に同意

4. 言語とエディションを選択
   - 言語: English (United States) を選択
     - 日本語パスは開発ツール（Node.js/npm、Python/pip、Docker など）で問題を起こすことがあるため
     - 英語でインストールするとシステムフォルダが英語パスで初期化され、開発環境の安定性が向上する
   - 「この PC におすすめのオプションを使う」のチェックを外す

5. メディアの種類を選択
   - 「USB フラッシュ ドライブ」を選択

6. USB メモリを選択
   - 対象の USB メモリを選択（内部データは消去される）

7. 書き込み完了を待つ
   - 最新の Windows 11 がダウンロードされ、USB に書き込まれる

## 2. Windows 11 のインストール

### USB から起動

1. 作成した USB メモリを PC に接続
2. PC を再起動し、BIOS/UEFI から USB 起動を選択

### インストール手順

#### 言語・キーボード選択

- Language: English (United States)
- Keyboard: Japanese

> **注意**: インストールメディアを English で作成したため、インストーラーも英語表示になる

#### クリーンインストール（重要）

1. 「カスタム: Windows のみをインストール（詳細）」を選択

2. ディスクのパーティションを **すべて削除**
   - EFI パーティション
   - MSR パーティション
   - Recovery パーティション
   - C ドライブ相当の領域
   - すべて削除して「未割り当て」にする

   > **警告**: この操作で PC 上のすべてのデータが消去される

3. 「未割り当て」を選択して「次へ」

4. インストール完了を待つ

## 3. 初回セットアップ（OOBE）

ローカルアカウントを作成し、`C:\Users\kou` を確保する。

### 手順

1. ネットワーク接続画面まで進む
   - 「ネットワークに接続しましょう」が表示されたら止まる

2. コマンドプロンプトを開く
   - `Shift + F10` を押す
   - Surface の場合: `Shift + Fn + F10`

3. 以下を入力して Enter

   ```
   OOBE\BYPASSNRO
   ```

4. 自動で再起動する（正常動作）

5. 再びセットアップを進める
   - ネットワーク画面で「インターネットに接続していません」を選択
   - 「制限された設定で続行」を選択

6. ローカルユーザーを作成
   - ユーザー名: `kou`
   - パスワード: 任意
   - セキュリティ質問: 適当に

この時点で `C:\Users\kou` が作成される。

## 4. Microsoft アカウントの紐づけ

### 手順

1. 設定 → アカウント → 自分の情報
2. 「Microsoft アカウントでのサインインに切り替える」
3. Microsoft アカウントでログイン

### 結果

- `C:\Users\kou` のまま（変わらない）
- OneDrive / Microsoft Store / ライセンスが有効になる

## 5. インストール後の処理

### Windows Update

- Windows Update をすべて完了させる
- 再起動が必要な場合は再起動

### 不要アプリのアンインストール

- プリインストールされた不要な Windows アプリを削除

### Microsoft Store の更新

- Microsoft Store を開き、すべてのアプリを更新

### 日本語言語設定

- 設定 → 時刻と言語 → 言語と地域
- 日本語の設定を確認・調整

> **ヒント**: Windows Update 完了後に言語設定を行うと早い
